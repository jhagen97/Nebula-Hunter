# Nebula-Hunter
Build a web-based, persistent Space RPG where players hunt bounties, upgrade their ships, and explore a galaxy of planets.
Backend Framework: Flask (Web Server)

Database: SQLAlchemy / SQL (Persistent Storage)

Frontend: Bootstrap (UI/UX)

Tools: Jira (Task Tracking) = https://jhagen1697.atlassian.net/jira/software/projects/NH/boards/2/backlog?atlOrigin=eyJpIjoiZTQ3MzMxODJiYWMzNDU0Yzk1ZGZkZWJmYjQzNjA2NjIiLCJwIjoiaiJ9
GitHub (Version Control) = https://github.com/jhagen97/Nebula-Hunter.git

How We Will Collaborate
We want to keep the code "decoupled." This means you can work on the game logic in a file called logic.py and test it in your own terminal. When your logic is ready, I will "plug it in" to the web server. This allows us to work at the same time without breaking each other's code.

The Roadmap (Phase 1: MVP)
Core Combat: Getting a basic "Attack vs. Health" function working.

The Galaxy: Defining the first set of planets and their difficulty.

Persistence: Setting up the database so you can "Save" your pilot.

Mission Control: Launching the first web dashboard where you can click "Hunt" and see the results.
